SiTeFL

SiTeFL is a Simple Text Formatting Language, with a text-to-html tool to go along with it. While it keeps some of markdown's features, it is quite minimal in its feature-set. To see a full list of features, check out the README in the GitHub repository.

You can also try the web version of sitefl at : SiTeFL online!

Implementation language : Go

Colon

Colon is a toy programming language that I am designing based on my learnings from an excellent book - "Writing interpreters in Go".

Implementation language : Go


Bru

Bru is an HDL based electronics simulator. It stitches together HDL files and input(script) files to produce a Go program, which when run, outputs the results of the simulation.

I've made a YouTube playlist on the basics of this tool. It can be found here.

Implemention language : Go


Dynar

Dynar is a C preprocessor of sorts that lets you use dynamic arrays as if they were an inbuilt datatype in C. It lets you create resizable arrays and provides functions to push, pop, insert and delete elements in/belonging to the array.

Implementation language : Java
